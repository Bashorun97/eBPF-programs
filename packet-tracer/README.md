### How to use packet-tracer

### Egress
Open one terminal or Tmux session
```
sudo python3 tracer.py
```

Open a second terminal or Tmux session
```
curl google.com
```

### Ingress
Open one terminal or Tmux session
```
python3 -m http.server
```

Open a second terminal or Tmux session
```
sudo python3 ingress
```
